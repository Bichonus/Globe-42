Fiche Projet Globe 42

Cette page décrit le projet réalisé dans le cadre du cours de Développement Web de la Majeure Informatique de MINES Saint-Étienne.
Ce projet consiste à réaliser un site Web pour l'association Globe 42, permettant aux membres de l'association d'avoir accès à une "fiche de médiation" pour chaque patient suivi.
